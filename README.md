## 🎯 Project Overview

This project is a **complete Network Traffic Analyzer and Bandwidth Management System** that helps network administrators monitor, analyze, and manage network traffic in real-time.

### Key Features

| Feature | Description |
|---------|-------------|
| **Real-time Traffic Collection** | Monitors bandwidth usage from all network devices |
| **Top Talkers Identification** | Identifies top bandwidth consumers |
| **Application Breakdown** | Shows which applications use the most bandwidth |
| **Device Utilization Monitoring** | Tracks per-device bandwidth usage |
| **Alert System** | Automatic alerts for high utilization |
| **QoS Recommendations** | Suggests traffic shaping policies |
| **Interactive Dashboard** | Live visualization of all metrics |

| Metric | Description | Alert Threshold |
